# 🎯 1. YOLOv8 Interactive Object Detection & Tracking

## 📌 Description

An interactive object detection and tracking system based on **YOLOv8 and ByteTrack**.

The system detects multiple objects in a video stream and allows the user to select any detected object by clicking on it. The selected object is continuously tracked while displaying a real-time spotlight crop.

This project demonstrates advanced computer vision concepts including:

- Object detection
- Multi-object tracking

- Human-computer interaction

# 📷 2. ESP32-CAM Object Detection Using YOLO

## 📌 Description

An embedded AI vision system combining an **ESP32-CAM wireless camera** with deep learning-based object detection.

The ESP32-CAM captures images and sends them through Wi-Fi to a computer, where AI inference is performed.

Two different implementations are provided:

### 🔹 YOLOv3 + OpenCV DNN

Uses the official Darknet YOLOv3 model files:

- `yolov3.cfg`
- `yolov3.weights`
- `coco.names`

These files must be downloaded manually from the official YOLO website.

### 🔹 cvlib Object Detection

A simplified implementation using the cvlib library.

Unlike the OpenCV DNN version, cvlib automatically manages the required YOLO model files.

## 🧠 Technologies

- ESP32-CAM
- Python
- OpenCV
- YOLOv3
- cvlib
- Wi-Fi Communication

## ⭐ Features

✅ Wireless AI camera system  
✅ Real-time object detection  
✅ ESP32 hardware integration  
✅ Multiple YOLO implementations  
✅ Computer vision processing on PC

# 🚗 4. Automatic Tunisian License Plate Recognition (ALPR)

## 📌 Description

A deep learning-based Automatic License Plate Recognition system specialized for **Tunisian vehicle license plates**.

The system combines:

1. YOLOv3 for license plate detection.
2. Image processing for character segmentation.
3. CNN-based OCR model for character recognition.

The system can detect and recognize license plates from images and real-time camera streams.

## 🧠 Technologies

- Python
- OpenCV
- YOLOv3 Darknet
- TensorFlow / Keras
- OCR
- Image Processing

## ⭐ Features

✅ Tunisian license plate detection  
✅ Automatic character segmentation  
✅ CNN-based OCR recognition  
✅ Real-time recognition capability  
✅ Tunisian plate format support
